## Get Started
1. Clone this repo
```shell script
git clone https://github.com/cleardusk/3DDFA_V2.git
cd 3DDFA_V2
```

2. Build Cython version of NMS 
```shell script
cd FaceBoxes
sh ./build_cpu_nms.sh
```

3. Run demos
```shell script
# running on still image
python3 demo.py -f examples/inputs/emma.jpg
```
